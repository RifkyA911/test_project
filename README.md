## Getting Started

First, Install All dependencies

```bash
npm install
```

Second, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

for the page list is:

1. `/` = datatable
2. `/products` = tampilan halaman product
